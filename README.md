Build project
-------------

Install Typescript compiler:

        sudo npm i -g typescript

Install node package manager:

        sudo apt-get install npm
        
Install project dependencies:

        npm install
        

        
Compile Typescript
-------------------

Run `grunt` command to automatically compile typescript on file change
